# 小红书内容合规检测规则

> 本文档为 xhs-publish 技能的合规检测参考。所有内容在发布前必须通过本规则检测。
> 检测策略：**生成时主动规避 → 生成后自动检测 → 违规自动重写 → 循环直到零违规**。

---

## 🔴 P0 红线（命中 → 自动重写 → 重新检测）

### 1. 诱导互动

小红书严禁通过利益交换或话术引导用户进行点赞/关注/评论/收藏等互动行为。

**关键词黑名单：**
```
双击有惊喜, 点赞看答案, 点赞领取, 关注即送, 关注领红包, 关注抽奖,
回复关键词获取, 评论区扣1, 收藏=白嫖, 先收藏以防找不到,
互赞, 互关, 互粉, 涨粉, 关注+点赞+评论
```

**模式匹配：**
- 「[点赞/关注/评论/收藏/转发] + [领/送/获取/免费/抽奖/解锁]」的组合
- 悬疑猎奇标题骗取互动（如「最后一张图惊了」「点开评论区」）

**安全替代写法：**
- ❌ 「关注我，下期更精彩」→ ✅ 「下一篇写什么？你来定」
- ❌ 「先收藏以防找不到」→ ✅ 「整理不易，觉得有用就存一下」
- ❌ 「评论区扣1领取资料」→ ✅ 「你怎么看？评论区聊聊」

---

### 2. 免费送福利引导

以利益（实物/虚拟物品/资源/优惠）交换用户互动行为。

**关键词黑名单：**
```
免费送, 抽奖, 领红包, 点赞领取, 关注领, 评论领, 转发领,
福利, 送福利, 粉丝福利, 白嫖, 薅羊毛
```

**模式匹配：**
- 「[互动行为] + [领取/赠送/免费获得]」的结构
- 「满X评论/点赞就送」的条件句

---

### 3. 站外导流

小红书对站外导流零容忍，涵盖正文、图片、评论区、私信。

**关键词黑名单：**
```
微信, VX, vx, 卫星, 威信, V❤, wei xin,
QQ, qq,
加我, 私聊, 后台找我, 看主页, +V,
淘宝, 天猫, 拼多多, 闲鱼, tb口令, 京东, JD,
抖音, 快手, B站
```

**谐音/变体检测：**
- 符号替代：V❤、薇❌、🔍搜+名字
- 拆字：wei xin、WéiXīn
- 暗语：「后台回复」「主页有」「简介有」「置顶评论」

---

### 4. 违禁词 / 绝对化用语

违反《广告法》的极限词和虚假承诺。

**绝对化用语黑名单：**
```
最好, 最佳, 最具, 最大, 最高, 最低, 最便宜, 最先进, 最受欢迎,
第一, 唯一, NO.1, TOP.1, TOP1, 独一无二, 一流, 全国第一,
国家级, 世界级, 顶级, 顶尖, 极品, 极致, 终极,
首个, 首选, 独家, 全国首发, 填补国内空白,
大牌, 金牌, 领袖品牌, 世界领先, 遥遥领先, 冠军, 之王
```

**虚假承诺黑名单：**
```
100%有效, 无效退款, 零风险, 特效, 强效, 速效,
1天见效, 彻底清除, 根治, 永久, 万能, 无依赖,
纯天然, 无副作用
```

**安全替代写法：**
- ❌ 「最好的AI工具」→ ✅ 「我用过的AI工具里很推荐的一款」
- ❌ 「唯一的解决方案」→ ✅ 「一个很有效的解决方案」
- ❌ 「100%有效」→ ✅ 「亲测有效」或「实际体验下来效果不错」

## 🟡 P1 风险（命中 → 自动重写 → 重新检测）

### 5. 话题堆砌

**检测规则：** 正文中 `#` 话题标签数量 > 10 个
**处理：** 自动精简到 5-8 个，优先保留垂直精准标签

### 6. 诱导消费词

**关键词黑名单：**
```
抢疯了, 万人疯抢, 再不抢就没了, 全民免单,
秒杀, 点击领奖, 恭喜获奖, 限时免费
```
**处理：** 自动替换为客观描述

### 7. 权威性词语

**关键词黑名单：**
```
老字号, 中国驰名商标, 特供, 专供, 专家推荐,
质量免检, 免抽检
```
**处理：** 自动删除

### 8. 迷信用语

**关键词黑名单：**
```
招财进宝, 化解小人, 提升运气, 旺财, 算命, 风水
```
**处理：** 自动删除

### 9. 金融保证词

**关键词黑名单：**
```
收益率, 保底收益, 返本销售, 稳赚不赔, 无风险回报
```
**处理：** 自动删除或改为客观表述

### 10. 低质内容

**检测规则：** 正文 < 50 字且无实质信息量
**处理：** 自动扩充内容至 300 字以上

## 📋 其他高风险内容（人工判断）

以下内容类型需要人工判断，自动检测仅提示：

### AI 内容声明
- AI 辅助创作内容**须主动声明**（发布界面 → 设置 → 内容类型声明）
- 纯 AI 生成未标注 → 可能限流/违规判定

### 医疗健康
- 严禁承诺疗效（「无副作用」「根治」「一秒淡斑」）
- 普通账号严禁涉及医疗功效

### 未成年人保护
- 禁止涉及未成年人不当内容
- 禁止不良诱导（吸烟/饮酒等）

### 虚假营销
- 无真实体验的虚构测评
- 有商业利益关联却不报备
- 虚假人设（编造学历/收入/经历）

## 🔄 检测循环流程

```
生成内容
  ↓
全文扫描 P0+P1 关键词/模式
  ↓
有命中？──是──→ 自动重写违规部分
  │                    ↓
  │              再次全文扫描（回到上一步）
  │              （最多 3 轮）
  │
  └──否──→ ✅ 内容合规完成
```

**注意事项：**
- 每轮重写后必须**完整重新检测**全文，不能只检查上一轮修改的部分
- 最多重写 3 轮，如仍有违规则停止并报告
- 优先在生成阶段主动规避，检测循环是兜底保障
